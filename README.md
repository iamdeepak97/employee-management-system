# 🚀 Employee Management System (HRMS)

A full-stack Employee Management & Attendance Tracking System built using:

- 🔹 Django REST Framework (Backend)
- 🔹 React.js with Bootstrap (Frontend)
- 🔹 MySQL (Database)

---

## 📌 Project Overview

This system allows organizations to:

- Manage employees
- Track daily attendance
- View attendance records employee-wise
- Filter attendance
- View dashboard counters
- Analyze present/absent statistics

This project is built with a production-ready structure and clean UI design.

---

## 🏗 Tech Stack

### Backend
- Python
- Django
- Django REST Framework
- MySQL

### Frontend
- React (Vite)
- Bootstrap 5
- Axios

---

## ✨ Features

### 👨‍💼 Employee Management
- Add Employees
- Unique Employee ID
- Department Management
- Email Validation

### 🕒 Attendance Management
- Mark Present / Absent
- One attendance per employee per day
- Filter by employee
- View attendance history

### 📊 Dashboard
- Total Employees Counter
- Total Attendance Records
- Present Count
- Absent Count
- Responsive UI
- Modern Bootstrap Cards
